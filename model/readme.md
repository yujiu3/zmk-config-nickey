# ケースモデル

手元のBambu Lab P1Sの0.4mmノズル, 0.12mmピッチ での確認は出来ています。

## トッププレート

- NickeyLtop.stl
- NickeyRtop.stl


## ボトムプレート

- NickeyL_bottom.stl
- NickeyR_bottom.stl

右側のボトムプレートです。

## マイコンカバー

- xiaoCover.stl

XiaoBLEのカバーです。リセットスイッチの棒も同一モデルに含まれているのでBambuStudioの分割を実施すると別々に印刷できます。

## 電源スイッチ

- powerswitch.stl

## トッププレート(option)

- NickeyLtopAAA
- NickeyRtopAAA

単4電池Modをしたい人はご活用下さい。

## PCB

- NickeyL_PCB.stl
- NickeyR_PCB.stl

ケースを自作したい人はご活用下さい。
